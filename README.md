## 🧠 Overview

PolkaArena is a decentralized application built for the **DoraHacks PolkaArena Hackathon**, combining:

* 🤖 AI-powered processing
* 🌐 IPFS decentralized storage
* 🔗 Blockchain integration (EVM / PVM-ready)

The goal is to create a **secure, scalable, and transparent system** for handling user data and intelligent workflows.

---

## ❗ Problem Statement

Modern applications suffer from:

* Centralized data storage 🏢
* Privacy and security risks 🔓
* Lack of transparency ❌

AI applications worsen this by processing sensitive user data without decentralization.

---

## 💡 Solution

PolkaArena solves this by integrating:

* **AI models** → intelligent processing
* **IPFS** → decentralized storage
* **Blockchain** → trust & immutability

✅ Result:

* Secure data handling
* No single point of failure
* Transparent architecture

## 🧪 Run Locally

```bash
# Clone repo
git clone https://github.com/your-username/polka-arena.git

# Go into project
cd polka-arena

# Install dependencies
npm install

# Add environment variables
touch .env.local
```

### 🔐 Environment Variables

```env
PINATA_API_KEY=your_key
PINATA_SECRET_API_KEY=your_secret
NEXT_PUBLIC_API_URL=http://localhost:3000
```
